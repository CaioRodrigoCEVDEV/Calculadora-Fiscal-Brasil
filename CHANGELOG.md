# Changelog

Todas as mudanças relevantes deste projeto serão documentadas neste arquivo.

O formato segue uma adaptação de Keep a Changelog e o projeto usa versionamento semântico.

## [1.0.0] - 2026-06-20

### Adicionado

- Publicação inicial da Calculadora Fiscal Brasil.
- Home pública do projeto.
- Calculadora de ICMS.
- Calculadora de ICMS-ST.
- Memória de cálculo.
- Área explicativa “Entenda o cálculo”.
- Botões de apoio para copiar e compartilhar cálculo, se já existirem no projeto.
- Arquivos públicos essenciais para SEO, quando existentes:
- `sitemap.xml`
- `robots.txt`
- `ads.txt`
- Identidade visual inicial com logo e favicon.

### Observações

- Primeira versão pública publicada em produção.
- Projeto preparado para evolução com novas calculadoras fiscais, melhorias de SEO e monetização via AdSense.
